Claude → GitHub → Notion 作業フロー

🤖
① Claude でHTML生成
claude.ai アプリ
  • HTMLを生成・表示
  • コードを長押し → コピー
🐙
② GitHub アプリでファイル作成
GitHub iOS アプリ
  • 上位リポジトリ → notion-embed
  • コード → … → Create new file
  • ファイル名入力 → HTMLをペースト
  • Commit で保存
⚙️
③ GitHub Pages 設定
Safari ブラウザ（必須）
  • Settings → Pages
  • Branch: main → Save
  • URLが発行される
⚠ GitHubアプリ非対応 → Safariで実施
📝
④ Notion にコードを貼り付け
Notion アプリ
  • /code → Code ブロック挿入
  • 言語: HTML に設定
  • ①のHTMLをペースト
✓ 完了
